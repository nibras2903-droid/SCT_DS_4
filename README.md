## Analyze Traffic Accident Data to Identify Patterns Related to Road Conditions, Weather, and Time of Day
Visualize Accident Hotspots and Contributing Factors

---

## 📌 Project Description

This project was completed as part of the **SkillCraft Technology Data Science Internship – Task 04**.

The objective of this project is to analyze a real-world traffic accident dataset to identify patterns associated with road conditions, weather, lighting, and time of day. Through Exploratory Data Analysis (EDA) and data visualization, the project uncovers accident hotspots and key contributing factors that influence accident frequency and severity.

---

## 🎯 Objective

- Analyze traffic accident data.
- Identify patterns related to:
  - Road Conditions
  - Weather Conditions
  - Time of Day
  - Lighting Conditions
  - Accident Severity
- Visualize accident hotspots.
- Discover major contributing factors through data analysis.
